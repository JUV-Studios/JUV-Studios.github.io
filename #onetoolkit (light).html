Jaiganésh 2021-11-18 08:59:15 PM
New ThemeService API for overriding app theme easily! How it works: We provide a property where you set the theme to override (Default, Light or Dark), by default set to Default. In XAML apps (WinUI 2.x and WinUI 3), changing the theme override will automatically change the requested theme of the XamlRoot which will apply to all controls where you haven't overridden the theme manually. Additionally there's a AppThemeChanged event which is raised when the app theme changes (also includes system theme changes when you've not override the theme) so that you can apply any custom customisation such as changing the title bar colour or manually change the theme that your UI is rendered in if not a XAML app. The AppThemeChanged provides you the new theme in terms of Light or Dark (not including Default) so the code that changes the theme doesn't need to bother with system default or a custom theme.
